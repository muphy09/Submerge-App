# Quick Start Guide

## Get Started in 3 Steps

### 1. Install Dependencies
Open your terminal in the `pool-proposal-app` folder and run:

```bash
npm install
```

This will install all required packages (React, Electron, TypeScript, etc.)

### 2. Start the Application
```bash
npm run dev
```

This command will:
- Start the development server
- Open the Pool Proposal Manager desktop application
- Load sample data (pool models, equipment, pricing)

### 3. Create Your First Proposal

When the app opens:
1. Click **"Create New Proposal"**
2. Fill in customer information
3. Use the navigation tabs to complete each section
4. Click **"Submit Proposal"** when done
5. View, print, or export to PDF!

## Sample Data Included

The app comes pre-loaded with:
- 8 pool models (fiberglass, concrete, vinyl)
- 16 equipment items (pumps, filters, heaters, cleaners)
- 12 water features (jets, fountains, waterfalls)
- Reference pricing for all materials

## Common Tasks

### View All Proposals
Home screen shows all saved proposals with status and total cost

### Edit a Proposal
1. Click on proposal card from home screen
2. Click "Edit Proposal" button
3. Make changes and save

### Export to PDF
1. Open any proposal
2. Click "Export PDF"
3. File saves to your Downloads folder

### Print Proposal
1. Open any proposal
2. Click "Print"
3. Select your printer and print
